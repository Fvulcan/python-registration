# Python Registration


This project was part of my Python fundamentals qualification in programming basics.

The goal was not to build the build application from zero, but to complete an existing Python starter project by implementing the required TODO functions.

I implemented:
- normalize_name()
- parse_birth_year()
- format_profile()
- parse_email() as an optional bonus function

The main focus was input validation, clean data formatting, and proper error handling using ValueError.
